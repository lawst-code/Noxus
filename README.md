## Noxus CLI

A basic CLI tool for plugin management.

### Installation

For development, install in editable mode:

```bash
cd noxus-cli
pip install -e .
```

### Usage

```bash
# Initialize a new plugin
python noxus.py init my-plugin-name

# Show help
python noxus.py --help
python noxus.py init --help
```
